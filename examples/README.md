# Examples

이 디렉토리는 Game AI Harness의 예시 산출물을 보관한다.

## design-systems/kernel-terminal

`kernel-terminal`은 `design-system-spec` skill로 만들 수 있는 디자인 시스템 산출물 예시다.

주의:

- 이 폴더는 하네스 필수 런타임 파일이 아니다.
- `skills/`에 있는 공식 skill과 다르다.
- 대상 게임 프로젝트에 자동 동기화되지 않는다.
- 실제 프로젝트에서는 장르와 아트 방향에 맞는 별도 디자인 시스템을 생성해야 한다.
- 폰트 파일은 라이선스 확인 후 프로젝트에서 직접 준비한다 (예시 저장소에 바이너리 폰트를 기본 포함하지 않음).
