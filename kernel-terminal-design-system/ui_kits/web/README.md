# Web UI Kit (Kernel marketing + docs)

`index.html` 은 **kernel.dev** 의 marketing landing + docs 미리보기를 한 화면에 담은 클릭 가능한 프로토타입입니다.

## Structure
- Top nav bar (logo, products, docs, github, sign-in)
- Hero with terminal demo block
- Feature grid (3 columns, squared cards)
- Code sample block (syntax-highlighted by classname)
- Docs sidebar + content
- Footer

All components are inlined in a single HTML file (matches the terminal kit pattern). Edit directly.
